# PlaudExporter
Sign in to Plaud, see Transcript/Summary/Notes/Audio availability per recording at a glance, and bulk-download what you pick — in parallel, to a folder you choose.

## Requirements
- Windows 10 or 11 (64-bit).
- .NET Framework 4.x — present by default on Win10/11
- A default web browser, for the Plaud sign-in (OAuth opens in your browser).
- Internet access to web.plaud.ai, platform.plaud.ai, and *.amazonaws.com 
- TCP port 8199 free on localhost during sign-in 
- A Plaud account.
- No admin rights
